# Dashboard

Power BI dashboard with 4 stakeholder tabs built from the SQL queries in this repo.

## Tabs

**Tab 1 : Uber CEO**
Platform health overview. Take rate by fare decile, margin concentration in D10, holiday demand cliff, TLC violation exposure.

**Tab 2 : Lyft CEO**
Competitive positioning. Borough performance, growth quality scores, Late Night risk segments to avoid.

**Tab 3 : NYC TLC Commissioner**
Regulatory oversight view. Driver pay compliance by borough and time, congestion fee burden by trip distance, enforcement targeting.

**Tab 4 : New Market Entrant**
Strategic entry map. Zone opportunity matrix, airport paradox, WAV accessibility gap, recommended launch sequence.

## Data Sources

All visuals pull from the CSV files in the `/data` folder, which are query outputs from the SQL files in `/sql`.
